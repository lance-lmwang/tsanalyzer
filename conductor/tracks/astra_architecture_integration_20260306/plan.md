# Implementation Plan: Astra Architecture Integration

This plan details the step-by-step approach to building the Lua pipeline engine.

## Step 1: Core Lua Runtime Initialization & Context (DONE)
- **Goal**: Establish the basic Lua environment embedded inside the C engine.
- **Tasks**:
  - Update `include/tsa_lua.h` and `src/tsa_lua.c` to wrap `luaL_newstate()`, `luaL_openlibs()`.
  - Create a global `tsa` module table in Lua to hold all factory functions.
  - Implement the CLI entry point `tsanalyzer run <script.lua>`.

## Step 2: Stream Object Binding (Input & Output)
- **Goal**: Expose basic network I/O to Lua.
- **Tasks**:
  - Create Lua userdata bindings for `tsa_source_t` (UDP/SRT input) and `tsp_handle_t` (UDP/SRT output).
  - Define metatables with `__gc` for safe destruction.
  - Example API to implement:
    ```c
    // C side: int lua_tsa_udp_input(lua_State *L)
    // Lua side: local input = tsa.udp_input("0.0.0.0:4000")
    ```

## Step 3: The Analyzer Binding & Pipeline Linking
- **Goal**: Expose the core `tsa_handle_t` (Analyzer) and implement the topological linking.
- **Tasks**:
  - Create Lua userdata bindings for `tsa_handle_t`.
  - Expose a method `set_upstream(source)` that essentially calls `tsa_stream_add_child()` at the C level.
  - Expose `join_pid(pid)` to handle selective routing.

## Step 4: Event Callbacks & Alarm Integration
- **Goal**: Allow Lua to react to events generated by the C metrology engine.
- **Tasks**:
  - Implement a registry in the C Lua context to hold references to Lua callback functions.
  - When `tsa_push_event()` is fired in C, cross the boundary to call the registered Lua function.
  - Example: `analyzer:on("alarm", function(event) print("Alarm triggered!") end)`

## Step 5: End-to-End Proof of Concept
- **Goal**: Write a complex `router.lua` script and verify it processes streams identically to the C gateway.
- **Tasks**:
  - Write `sample/router.lua` that takes 1 UDP input, analyzes it, and splits it into 2 UDP outputs based on PIDs.
  - Add this test to the standard integration test suite.
